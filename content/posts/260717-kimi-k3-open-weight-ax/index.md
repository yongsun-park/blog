## 들어가며

Moonshot AI가 Kimi K3를 발표했어요. [공식 발표](https://www.kimi.com/blog/kimi-k3)의 자체 평가에서는 최상위 상용 모델과 비교할 만한 성능을 보였고, 가장 강한 비공개 모델보다는 아직 뒤진다고 밝혔습니다.

2026년 7월 17일 현재 전체 모델 파일과 기술 보고서는 공개 전이에요. Moonshot은 모델 파일을 7월 27일까지 공개하고 기술 보고서도 함께 내놓을 예정이며, [공식 Hugging Face 모델 목록](https://huggingface.co/moonshotai/models)에는 아직 K3 모델 파일이 없습니다. 실제 업무 성능과 라이선스는 공개 뒤에 확인해야 해요.

---

## 고성능 Open Weight 선택지가 늘어납니다

일부 기업은 이미 [GLM-5.2](https://huggingface.co/zai-org/GLM-5.2) 같은 Open Weight 모델을 자체 환경에서 운영하고 있어요. K3가 공개되면 기업이 직접 운영할 수 있는 모델의 성능 기준이 프론티어급에 한 단계 더 가까워질 수 있다는 점이 기대됩니다.

---

## 4비트로 줄여도 모델 파일은 약 1.4TB입니다

K3에는 학습 과정에서 만들어진 숫자인 parameter가 2.8조개 있어요. MoE 구조는 896개 expert 그룹 가운데 token마다 16개만 골라 계산하지만, 입력에 따라 선택되는 그룹이 달라집니다. 계산에는 일부만 참여해도 896개 그룹 전체를 저장해야 해요.

모델 파일만 단순 계산하면 16비트에서 약 5.6TB, 4비트에서 약 1.4TB입니다. 2.8조 parameter에 bit 수를 곱한 산술 추정일 뿐, 측정된 최소 GPU 메모리는 아니에요. 실제로 실행할 때는 양자화 정보와 입력 처리, 여러 요청을 위한 추가 메모리가 더 필요합니다.

---

## 공개 전이라 8비트 기준으로 계산해 봤습니다

아직 4비트 모델 파일의 형식과 실행 엔진 호환성, 업무 품질을 확인할 수 없어서 장비 계획은 보수적으로 8비트를 기준으로 잡았어요. 이 경우 모델 파일은 약 2.8TB이고, 실행 여유까지 포함한 계획 용량은 3.5~4.0TB 정도입니다. 공식 최소 사양이나 실측값이 아니라 산술과 운영 가정이에요.

이 계획 용량을 메모리에 올리는 최소선은 H200 32개(8-GPU 서버 4대, 4.512TB)나 B300 16개(서버 2대, 4.608TB) 정도예요. ([NVIDIA HGX 구성표](https://docs.nvidia.com/enterprise-reference-architectures/hgx-ai-factory/latest/components.html)) 여러 사용자의 속도와 동시 사용을 보장하는 구성은 아닙니다. expert 그룹 사이의 빠른 통신까지 고려해 Moonshot은 64개 이상의 가속기를 권장해요.

공개될 4비트 모델의 품질과 호환성이 괜찮다면 B300 8개가 들어간 서버 한 대에 올릴 가능성도 있어요. 이 서버의 공식 GPU 메모리는 2.3TB이고 4비트 모델 파일은 산술상 약 1.4TB입니다. 다만 실행에 필요한 추가 메모리와 실제 속도, 동시 사용자 수는 공개 뒤에 측정해야 해요.

---

## B300 한 대가 부서 사용량을 감당한다면

서버 한 대가 부서 사용량을 감당한다면 대기업 연구소나 사업부가 10억원 안팎에서 자체 AI 환경을 검토할 수 있을 것 같아요. 정확한 가격은 서버 구성과 네트워크, 지원 계약을 포함한 견적을 받아봐야 합니다. 개인에게는 여전히 어려운 규모지만 기업의 단위 조직에는 검토 가능한 투자 범위가 될 수 있어요.

사용량이 적다면 ChatGPT나 Claude 같은 외부 서비스가 더 저렴하고 운영도 편할 수 있어요. 자체 구축은 모델과 데이터를 직접 통제하는 대신 초기 투자와 운영 부담이 생깁니다. 보안과 데이터 통제는 자체 구축을 고르는 여러 기준 중 하나이고, 직접 운영하더라도 접근 권한과 사용 기록, 장애 대응은 따로 갖춰야 해요.

---

## 마무리

모델이 공개되면 4비트 실제 업무 성능과 기업이 사용할 수 있는 라이선스, B300 한 대의 처리 속도와 첫 응답 시간, 동시 사용자 수를 확인해 보고 싶어요. 기대한 수준이라면 기업이 직접 운영할 고성능 Open Weight 선택지가 하나 더 늘어날 것 같습니다.

B300 8장이 들어간 서버 한 대쯤 집에 있으면 좋겠네요. 전기세는 감당하기 어렵겠지만요.
